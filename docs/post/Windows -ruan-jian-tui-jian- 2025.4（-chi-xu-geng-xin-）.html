Windows 软件推荐 2025.4（持续更新）

前言

其实我本来不想做这个的，因为网上已经有太多了。直到我看到我妈那个赛博养蛊的电脑……
好了，正片开始

软件推荐

办公

这里推荐一个精简过的 WPS Office，如果要做 PPT 可以安装 iSlide 插件。之所以不推荐 Microsoft Office，是因为它只能安装在 C 盘，不利于磁盘整洁

输入法

这里推荐一个魔改过的搜狗输入法，精简去广

解压缩

解压缩软件，大家最熟悉的当然是 WinRAR
然鹅，广告和弹窗实在太多，解压缩速度也不尽人意。所以这里建议使用 Bandizip + NanaZip
为什么要推荐两款呢？Bandizip 压缩率非常低，但是解压极度迅速；而 NanaZip 压缩率非常高，但是解压缩速度很慢，正好互补

视频播放

视频播放这里只推荐 Potplayer。没什么好多说的，太强大了！（建议搭配 madVR 食用 配置教程）

应用商店

这里不推荐用 Microsoft Store，建议使用第三方替代品 UniGetUI

图片查看

不说了，Imageglass，用了一年了，开源又自定义性强

下载器

这里只推荐 Gopeed，不推荐从官网下载，没有便携版。Neat Download Manager 不知道为什么我有时会闪退

卸载器

Bulk Crap Uninstaller，批量卸载非常好用

文件搜索

Everything，不推荐用自带的，是因为慢的跟蜗牛一样，这个几乎秒出

PDF

UPDF，比 Adobe DC 好用

杀毒软件

这里只推荐卡巴斯基，配置教程 360个人用下来感觉太敏感

截图

Snipaste 可以使你方便地截图

GIF

ScreenToGif 可以让你将屏幕里一部分录制为 GIF

其他

PowerToys 微软出品的实用小工具
图吧工具箱 最纯净的硬件工具箱
火绒小工具提取版 提取码：9wuj，装完卡巴斯基必下这个

后话

本文章持续更新，有推荐的其它纯净软件也可以留言
